Title: HostelHub one stop solution for all students
Description of the heck: This website mainly for hostlers who are residing in the hostels can raise their complaints regarding electricity,mess,cleanliness etc and authority seemlessly address the complaints.
Tech stack used:
Frontend -> React js, css, Bootstrap
Backend -> Express.js, Nodejs
Database -> MongoDB Atlas
libraries -> Mongoose , shadcn

Dependencies ->  {
    "all": "^0.0.0",
    "cors": "^2.8.5",
    "express": "^4.19.2",
    "mongoose": "^8.3.1",
    "nodemon": "^3.1.0",
    "react-router-dom": "^6.22.3",
}
Installation steps ->
1. Clone the repository.
2. Go to the Client and Server directory in two different terminals using cd Client and cd Server
3. Then do npm install in client side
4. Then do npm start in both client and server and project will be visible
